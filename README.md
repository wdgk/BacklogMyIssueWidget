# Backlog My Issues Widget

macOS メニューバーに常駐し、自分に割り当てられた Backlog の課題をフローティングパネルで一覧表示するデスクトップウィジェットです。

## 機能

- **メニューバー常駐** — メニューバーアイコンからワンクリックで課題一覧を表示
- **フローティングパネル** — 常に最前面に表示されるドラッグ可能なパネル（全 Spaces 対応）
- **自動更新** — 1 時間ごとに課題を自動取得
- **期限超過の強調表示** — 期限切れの課題を赤色で強調
- **ステータス・優先度の可視化** — 色分けされたバッジとプライオリティバーで一目で把握
- **課題への直接アクセス** — クリックで Backlog の課題ページをブラウザで開く
- **キーボードショートカット** — パネル表示 / 更新 / 終了をキーボードで操作
- **セキュアな API キー保存** — macOS Keychain に API キーを安全に保存

## 動作環境

- macOS 26.0 (Tahoe) 以降
- Xcode（ビルドする場合）

## インストール

[Releases](https://github.com/wdgk/BacklogMyIssueWidget/releases) から zip をダウンロードし、解凍した `BacklogMyIssueWidget.app` をアプリケーションフォルダに移動してください。

> **Gatekeeper の警告が表示される場合**
>
> このアプリは Apple Developer ID で署名されていないため、初回起動時に警告が表示されます。
> **システム設定** → **プライバシーとセキュリティ** → **「このまま開く」** をクリックして許可してください。

## セットアップ

### 1. ビルド（ソースから）

```bash
git clone <repository-url>
cd backlog-my-issue-widet/BacklogMyIssueWidget
open BacklogMyIssueWidget.xcodeproj
```

Xcode でプロジェクトを開き、ビルド＆実行（`Cmd + R`）してください。

### 2. Backlog API キーの取得

1. Backlog にログイン
2. **個人設定** → **API** に移動
3. 新しい API キーを発行

### 3. アプリの設定

1. アプリを起動するとメニューバーにアイコンが表示されます
2. フローティングパネルの **歯車アイコン** をクリック
3. 以下を入力して保存:
   - **スペース URL** — `myspace.backlog.com` または `myspace.backlog.jp`
   - **API キー** — 取得した API キー

## 使い方

### 基本操作

- メニューバーアイコンをクリックしてメニューを表示
- パネルはドラッグで好きな位置に移動可能（位置は自動保存）
- 課題をクリックすると Backlog の該当ページが開きます

### キーボードショートカット

| キー | 操作 |
|------|------|
| `⌘W` | ウィジェットの表示 / 非表示 |
| `⌘R` | 課題を手動更新 |
| `⌘Q` | アプリを終了 |

### 表示される課題

ステータスが **未対応**・**処理中**・**処理済み** の自分の課題が、期限の近い順に表示されます（最大 100 件）。
